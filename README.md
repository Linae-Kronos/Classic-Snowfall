# Classic-Snowfall

Accelerates key bindings so that they are activated by key press rather than key release

Supported Actionbars
* Bartender
* Bongos 2
* Blizzard Actionbars

# Commands
```
/csselfcast    Toggle Alt Self Cast support
```
